# Hearth Hybrid Assistant

This version moves beyond a single canned scenario.

## What it includes
- Warm tablet-first visual identity
- Starter cards for common senior tech issues
- Freeform natural-language input
- Upload tray for photos or screenshots
- Plain-language guided flows
- "Explain it another way" option
- Visual "show me where" support
- Live helper handoff panel
- Copyable support note

## Important
This is still a front-end prototype for GitHub Pages. It demonstrates the product experience and routing logic.
In a production version, the conversation and image understanding would be connected to the OpenAI API on a backend.
