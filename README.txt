# Willow Assist

Willow Assist is a premium front-end prototype for a conversation-first tech support experience designed for older adults, caregivers, and staff.

## Why this version is stronger
This rebuild moves away from a basic dashboard or generic AI wrapper. It is designed to feel like a real product someone would buy:
- branded identity
- emotional, calm conversation flow
- real-life scenario cards instead of technical categories
- live helper handoff panel
- accessibility controls
- polished consumer-style UI

## Files
- index.html
- styles.css
- app.js
- submission_copy.txt
- pitch_points.txt

## How to use
Open `index.html` in a browser, or publish the files to GitHub Pages.

## Best live demo flow
1. Open the “I can’t hear my grandson on a video call” scenario
2. Click through one or two responses
3. Show the “I’m confused” button
4. Point out the live helper handoff panel
5. Copy the support note

## Strong product angle
This is not just an AI chatbot. It is a calmer digital support companion built around trust, accessibility, and real support workflows.
