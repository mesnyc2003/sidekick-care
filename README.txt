# Sidekick Care – Premium Product Concept

This is a premium front-end prototype for **Sidekick Care**, an AI tech support product designed for older adults, caregivers, and support teams.

## What changed from the starter
This version was rebuilt to feel like a real high-value product:
- premium landing page
- stronger product positioning
- older adult / caregiver / staff modes
- accessibility controls
- polished guided demo
- structured escalation summary
- stronger narrative for usefulness, execution, and polish

## Files
- index.html
- styles.css
- app.js
- submission_copy.txt
- pitch_points.txt

## How to use
Open `index.html` in a browser.

## Product story
This product is based on real-world experience helping older adults with common technology problems. It focuses on calm guidance, accessibility, and better handoffs when support is still needed.
